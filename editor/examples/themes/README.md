# Theme examples

Copy any of these to `~/.config/lighteditor/theme.toml` (XDG / Linux) or
`~/Library/Application Support/lighteditor/theme.toml` (macOS). The editor
watches the file and reloads on save — you can edit colors with the editor
itself and see the change land within a frame.

Every field is optional: a theme that only sets `[syntax].keyword` keeps
the default for everything else. Hex strings accept `#RGB`, `#RGBA`,
`#RRGGBB`, and `#RRGGBBAA`.

| File                    | Palette                                                 |
|-------------------------|---------------------------------------------------------|
| `solarized-dark.toml`   | Ethan Schoonover's classic dark Solarized               |
| `solarized-light.toml`  | Light variant of the same palette                       |
| `monokai.toml`          | Bright, high-contrast green / pink                      |
| `gruvbox-dark.toml`     | Warm retro browns + ochre highlights                    |
| `nord.toml`             | Cool icy blue Nordic palette                            |
| `tokyo-night.toml`      | Modern dark purple / cyan                               |
